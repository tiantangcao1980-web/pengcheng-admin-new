## 一、项目概览

### 1.1 项目定位
**MasterLife** 是一个 AI 驱动的房地产智能协作平台，采用前后端分离 + 模块化单体架构。

### 1.2 技术架构
| 层级 | 技术栈 | 状态 |
|------|--------|------|
| 后端 | Spring Boot 3.2.2 + MyBatis-Plus 3.5.5 + Sa-Token | ✅ 成熟稳定 |
| AI 引擎 | Spring AI Alibaba 1.0.0.2 + AgentScope | ✅ 已集成 |
| 前端 PC | Vue 3.4 + Naive UI 2.37 + TypeScript | ✅ 成熟稳定 |
| 移动端 | UniApp + uView Plus 3.3 | ✅ 已集成 |
| 数据库 | MySQL 8.0 + PostgreSQL 16 (pgvector) | ✅ 已配置 |
| 缓存 | Redis 7.0+ | ✅ 已配置 |
| 对象存储 | MinIO / 阿里云 OSS | ✅ 已集成 |
| 文件预览 | kkFileView 4.4.0 | ✅ 已集成 |
| 在线编辑 | OnlyOffice | ⚠️ 待部署验证 |
