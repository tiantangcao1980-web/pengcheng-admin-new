## 八、部署步骤

### 8.1 快速部署

```bash
# 1. 数据库迁移
mysql -u root -p pengcheng-system < sql/pengcheng-system.sql
# 或按顺序执行 V1-V35 迁移脚本

# 2. 配置环境变量
cp .env.example .env
# 编辑 .env 配置数据库、Redis、AI 密钥等

# 3. 启动后端
cd pengcheng-admin
docker-compose up -d

# 4. 部署前端
cd pengcheng-ui
npm run build
# 将 dist 目录部署到 Nginx

# 5. 验证访问
# 前端：http://localhost:5173
# API 文档：http://localhost:8080/api/doc.html
```
