## 五、部署方案

### 5.1 Docker 一键部署（推荐）

```bash
# 1. 克隆项目
git clone <repo-url>
cd pengcheng-admin

# 2. 配置环境变量
cp .env.example .env
# 编辑 .env 配置密码

# 3. 启动服务
docker-compose up -d

# 4. 验证
docker-compose ps

# 5. 访问
# 前端：http://localhost
# API: http://localhost:8080/api/doc.html
```

### 5.2 服务器要求

| 配置 | 最低 | 推荐 |
|------|------|------|
| CPU | 4 核 | 8 核 |
| 内存 | 8GB | 16GB |

| 带宽 | 5Mbps | 10Mbps |
