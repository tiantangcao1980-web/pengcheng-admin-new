## 一、电脑端安装包

### 方案一：PWA 离线应用（推荐 ⭐⭐⭐⭐⭐）

**优点**:
- ✅ 无需额外打包，直接复用 Web 版本
- ✅ 自动更新
- ✅ 跨平台（Windows/macOS/Linux）
- ✅ 支持离线使用
- ✅ 可安装到桌面

**步骤**:

#### 1. 添加 PWA 支持

```bash
cd pengcheng-ui
npm install vite-plugin-pwa -D
```

#### 2. 配置 Vite

```typescript
// vite.config.ts
import { VitePWA } from 'vite-plugin-pwa'

export default defineConfig({
  plugins: [
    vue(),
    VitePWA({
      registerType: 'autoUpdate',
      includeAssets: ['favicon.ico', 'apple-touch-icon.png', 'masked-icon.svg'],
      manifest: {
        name: 'MasterLife',
        short_name: 'MasterLife',
        description: 'MasterLife - 高效沟通，智慧办公',
        theme_color: '#18a058',
        background_color: '#ffffff',
        display: 'standalone',
        icons: [
          {
            src: '/pwa-192x192.png',
            sizes: '192x192',
            type: 'image/png'
          },
          {
            src: '/pwa-512x512.png',
            sizes: '512x512',
            type: 'image/png'
          },
          {
            src: '/pwa-512x512.png',
            sizes: '512x512',
            type: 'image/png',
            purpose: 'any maskable'
          }
        ]
      },
      workbox: {
        globPatterns: ['**/*.{js,css,html,ico,png,svg,woff2}'],
        runtimeCaching: [
          {
            urlPattern: /^https:\/\/api\./i,
            handler: 'NetworkFirst',
            options: {
              cacheName: 'api-cache',
              networkTimeoutSeconds: 10,
              cacheableResponse: {
                statuses: [0, 200]
              }
            }
          }
        ]
      }
    })
  ]
})
```

#### 3. 构建

```bash
npm run build
```

#### 4. 部署

将 `dist` 目录部署到服务器，用户访问后浏览器会提示安装。

#### 5. 用户安装步骤

1. 使用 Chrome/Edge 访问网站
2. 浏览器地址栏出现安装图标
3. 点击安装，创建桌面快捷方式
4. 可离线使用部分功能

### 方案二：Electron 打包（可选 ⭐⭐⭐）

**优点**:
- ✅ 真正的桌面应用
- ✅ 可访问系统 API
- ✅ 更好的离线体验

**缺点**:
- ⚠️ 需要额外配置
- ⚠️ 包体积较大（约 100MB+）
- ⚠️ 需要单独维护

**步骤**:

#### 1. 安装 Electron

```bash
cd pengcheng-ui
npm install --save-dev electron electron-builder concurrently wait-on
```

#### 2. 创建 Electron 主进程

```javascript
// electron/main.js
const { app, BrowserWindow } = require('electron')
const path = require('path')

function createWindow() {
  const win = new BrowserWindow({
    width: 1400,
    height: 900,
    webPreferences: {
      nodeIntegration: false,
      contextIsolation: true
    },
    icon: path.join(__dirname, '../public/favicon.ico')
  })

  // 开发环境加载 Vite 开发服务器
  if (process.env.NODE_ENV === 'development') {
    win.loadURL('http://localhost:5173')
    win.webContents.openDevTools()
  } else {
    // 生产环境加载构建后的文件
    win.loadFile(path.join(__dirname, '../dist/index.html'))
  }
}

app.whenReady().then(createWindow)

app.on('window-all-closed', () => {
  if (process.platform !== 'darwin') app.quit()
})
```

#### 3. 配置 package.json

```json
{
  "name": "masterlife-desktop",
  "version": "1.0.0",
  "main": "electron/main.js",
  "scripts": {
    "electron:dev": "concurrently \"npm run dev\" \"wait-on http://localhost:5173 && electron .\"",
    "electron:build": "npm run build && electron-builder",
    "electron:preview": "electron ."
  },
  "build": {
    "appId": "com.pengcheng.masterlife",
    "productName": "MasterLife",
    "directories": {
      "output": "release"
    },
    "files": [
      "dist/**/*",
      "electron/**/*"
    ],
    "win": {
      "target": ["nsis"],
      "icon": "public/favicon.ico"
    },
    "mac": {
      "target": ["dmg"],
      "icon": "public/favicon.ico"
    },
    "linux": {
      "target": ["AppImage"],
      "icon": "public/favicon.ico"
    }
  }
}
```

#### 4. 打包

```bash
# Windows
npm run electron:build

# macOS
npm run electron:build -- --mac

# Linux
npm run electron:build -- --linux
```

#### 5. 输出文件

```
release/
├── MasterLife-Setup.exe      # Windows 安装包
├── MasterLife.dmg            # macOS 安装包
└── MasterLife.AppImage       # Linux 安装包
```

### 方案三：Web 应用（最简单 ⭐⭐⭐⭐）

**直接使用 Web 版本，无需打包**

**步骤**:
```bash
# 1. 构建
npm run build

# 2. 部署到 Nginx
server {
    listen 80;
    server_name masterlife.example.com;
    root /var/www/masterlife;
    index index.html;

    location / {
        try_files $uri $uri/ /index.html;
    }

    location /api {
        proxy_pass http://localhost:8080;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }
}
```

**用户访问**: 打开浏览器输入网址即可使用

## 二、移动端安装包

### 方案一：Android APK（推荐 ⭐⭐⭐⭐⭐）

**使用 HBuilderX 云打包**

#### 1. 准备工作

- 注册 DCloud 开发者账号：https://dev.dcloud.net.cn/
- 下载 HBuilderX：https://www.dcloud.io/hbuilderx.html

#### 2. 配置 manifest.json

```json
{
  "name": "MasterLife",
  "appid": "__UNI__.xxxxxxxx",
  "description": "MasterLife - 高效沟通，智慧办公",
  "versionName": "1.0.0",
  "versionCode": "100",
  "app-plus": {
    "distribute": {
      "android": {
        "permissions": [
          "<uses-permission android:name=\"android.permission.INTERNET\"/>",
          "<uses-permission android:name=\"android.permission.ACCESS_NETWORK_STATE\"/>",
          "<uses-permission android:name=\"android.permission.ACCESS_WIFI_STATE\"/>",
          "<uses-permission android:name=\"android.permission.CAMERA\"/>",
          "<uses-permission android:name=\"android.permission.ACCESS_FINE_LOCATION\"/>",
          "<uses-permission android:name=\"android.permission.ACCESS_COARSE_LOCATION\"/>",
          "<uses-permission android:name=\"android.permission.READ_CONTACTS\"/>",
          "<uses-permission android:name=\"android.permission.WRITE_CONTACTS\"/>",
          "<uses-permission android:name=\"android.permission.RECORD_AUDIO\"/>",
          "<uses-permission android:name=\"android.permission.READ_PHONE_STATE\"/>",
          "<uses-permission android:name=\"android.permission.WRITE_EXTERNAL_STORAGE\"/>",
          "<uses-permission android:name=\"android.permission.READ_EXTERNAL_STORAGE\"/>"
        ],
        "minSdkVersion": 21,
        "targetSdkVersion": 30,
        "abiFilters": ["armeabi-v7a", "arm64-v8a"]
      },
      "icons": {
        "android": {
          "hdpi": "unpackage/res/icons/72x72.png",
          "xhdpi": "unpackage/res/icons/96x96.png",
          "xxhdpi": "unpackage/res/icons/144x144.png",
          "xxxhdpi": "unpackage/res/icons/192x192.png"
        }
      },
      "schemes": ["masterlife"]
    }
  }
}
```

#### 3. 云打包步骤

1. 打开 HBuilderX
2. 打开 `pengcheng-uniapp` 项目
3. 菜单：发行 → 原生 App-云打包
4. 选择"正式打包"
5. 等待打包完成（约 5-10 分钟）
6. 下载 APK 文件

#### 4. 输出文件

```
unpackage/dist/apk/
└── MasterLife_1.0.0.apk    # Android 安装包
```

#### 5. 用户安装步骤

1. 下载 APK 文件
2. 允许安装未知来源应用
3. 点击安装
4. 打开应用，授权必要权限

### 方案二：iOS IPA（需要 Apple 开发者账号 ⭐⭐⭐）

#### 1. 准备工作

- Apple 开发者账号（$99/年）
- macOS 电脑
- Xcode
- 证书和描述文件

#### 2. 配置 manifest.json

```json
{
  "app-plus": {
    "distribute": {
      "ios": {
        "bundleid": "com.pengcheng.masterlife",
        "version": "1.0.0",
        "appstore": true,
        "ipaResourceId": "__UNI__.xxxxxxxx",
        "devices": "universal",
        "privacyDescription": {
          "NSCameraUsageDescription": "需要使用相机进行拍照打卡",
          "NSLocationWhenInUseUsageDescription": "需要使用位置进行考勤打卡",
          "NSPhotoLibraryUsageDescription": "需要访问相册选择图片",
          "NSMicrophoneUsageDescription": "需要录音功能",
          "NSContactsUsageDescription": "需要访问联系人"
        }
      }
    }
  }
}
```

#### 3. 打包步骤

1. HBuilderX 菜单：发行 → 原生 App-云打包
2. 选择"iOS 应用 - App Store"
3. 上传证书和描述文件
4. 等待打包完成
5. 下载 IPA 文件

#### 4. 分发方式

**方式一：App Store 发布**
- 上传到 App Store Connect
- 提交审核
- 审核通过后发布

**方式二：企业签名分发**
- 需要企业开发者账号（$299/年）
- 使用第三方分发平台（如蒲公英）

**方式三：TestFlight 测试**
- 上传到 App Store Connect
- 添加测试用户
- 通过 TestFlight 安装

### 方案三：微信小程序（推荐 ⭐⭐⭐⭐）

#### 1. 准备工作

- 注册微信小程序账号
- 获取小程序 AppID

#### 2. 配置 manifest.json

```json
{
  "mp-weixin": {
    "appid": "wx_xxxxxxxxxxxxxxxx",
    "setting": {
      "urlCheck": false,
      "es6": true,
      "postcss": true,

    },
    "usingComponents": true,

      "scope.userLocation": {
        "desc": "你的位置信息将用于考勤打卡"
      }
    },
    "requiredPrivateInfos": [
      "getLocation",
      "chooseLocation"
    ]

#### 3. 打包步骤

1. HBuilderX 菜单：发行 → 微信小程序
2. 等待编译完成
3. 打开微信开发者工具
4. 导入生成的代码
5. 上传代码
6. 提交审核

#### 4. 输出目录

```
unpackage/dist/mp-weixin/
├── pages/          # 页面文件
├── static/         # 静态资源
├── app.js          # 小程序入口

## 五、常见问题

### Q1: 电脑端安装包在哪里下载？

**答**: 
- Web 版：直接访问公司部署的网址
- PWA 版：浏览器访问后点击安装
- Electron 版：联系管理员获取安装包

### Q2: 移动端 APK 安装失败？

**答**: 
1. 检查是否允许安装未知来源
2. 检查 APK 文件是否完整
3. 检查 Android 版本是否兼容（最低 Android 5.0）

### Q3: iOS 无法安装？

**答**: 
- 需要通过 App Store 下载
- 或使用 TestFlight 测试版
- 企业签名版本需要信任开发者

### Q4: 应用如何更新？

**答**: 
- Web/PWA 版：自动更新
- Electron 版：应用内检测更新
- Android：重新下载安装包
- iOS：App Store 更新

### Q5: 忘记密码怎么办？

**答**: 
- 联系公司管理员重置密码
- 或使用"忘记密码"功能自助找回
