## 一、产品定位

> **AI 驱动的中小企业智能协作平台**

一站式替代多套 SaaS 工具，主打 **「组织 + 客户 + AI 助手 + 移动办公」** 四大支柱，10–500 人企业当天可上线使用。

### 1.1 核心价值

| 价值 | 说明 |
|------|------|
| **一站式** | OA + CRM + AI + 移动办公四合一 |
| **AI 原生** | 每个业务模块均可由 AI Copilot 驱动（跟进提醒 / 文案生成 / 自动填单） |
| **全端可用** | Web 管理后台 + iOS/Android APP + 微信小程序，数据实时同步 |
| **开箱即用** | 私有化 / SaaS 双部署，企业一分钟开通 |

### 1.2 目标用户

| 角色 | 主阵地 | 核心场景 |
|------|--------|---------|
| **员工**（销售/顾问） | 手机 APP | 打卡 + 客户跟进 + AI 助手 |
| **管理层**（主管/老板） | APP + Web | 看数据 + 审批 + 设目标 |
| **管理员**（IT / HR） | Web | 组织 + 权限 + 配置 |

### 3.2 模块划分（Maven 多模块）

```
pengcheng-admin (parent)
├── pengcheng-common              # 通用基础（错误码 / TenantContext / I18n）
├── pengcheng-infra               # 基础设施
│   ├── pengcheng-db              # 数据访问 + 多租户拦截器
│   ├── pengcheng-redis           # 缓存
│   ├── pengcheng-oss             # 对象存储
│   ├── pengcheng-crypto          # 加密
│   ├── pengcheng-mail / sms / push / wechat / social / pay / auth / file / gen
├── pengcheng-core                # 业务核心
│   ├── pengcheng-system          # RBAC / 网关 / OpenAPI / SaaS / 自动化
│   ├── pengcheng-message         # 消息 / 通知 / IM
│   ├── pengcheng-hr / finance / crm / realty / ai / oa / bi / integration / job
└── pengcheng-api / pengcheng-starter
```

### 3.3 关键技术沉淀

1. **三大并行模式**：闭环 ①②③④⑤ 各自独立，14 个 Sub-Agent 并行交付
2. **SPI + Registry**：推送渠道（jpush / wechat / inbox / sms / app push）可插拔
3. **Feature Flag 默认关闭**：所有外部集成（微信支付 / 支付宝 / 第三方推送）出厂关闭，避免误用
4. **状态机 + 异步 + 失败隔离**：Webhook、域事件、自动化规则 — 失败不影响主流程
5. **SQL 注入 5 层防护**：BI 引擎参数化 / 关键字白名单 / 表级 ACL / 列级脱敏 / 审计日志
6. **签名验证三件套**：Webhook + e-Sign + OpenAPI 全部 HMAC-SHA256
7. **跨模块复用**：ApplicationContext.getBean 反射解循环依赖，业务模块不反向依赖 system

1. **基础设施** → docker-compose up -d mysql redis postgres minio

3. **可观测性** → docker-compose up -d prometheus loki grafana
4. **域名 / SSL** → nginx 反向代理（HTTPS）
5. **数据迁移** → Flyway 自动执行 V37-V76
6. **种子数据** → 默认管理员 / 角色 / 字典
7. **验证清单** → 跑完 5 大闭环冒烟测试
