## 1. 系统概述

鹏程房产销售管理系统是一套面向房地产企业的全流程销售管理平台，集客户管理（CRM）、项目管理、佣金结算、即时通讯、AI 智能辅助于一体。

### 1.1 核心功能

| 模块 | 功能 | 适用角色 |
|------|------|----------|
| 工作台 | 数据概览、AI 洞察、快捷入口 | 全员 |
| 客户管理 | 客户录入、跟进、判客、签约 | 销售顾问 |
| 项目管理 | 楼盘信息、房源管理、价格维护 | 项目经理 |
| 佣金管理 | 佣金计算、结算、审批 | 财务/管理层 |
| 即时通讯 | 单聊、群聊、@提及、消息搜索 | 全员 |
| 智能表格 | 自定义表格、看板、甘特图 | 全员 |
| AI 助手 | 智能对话、报表分析、文案生成 | 全员 |

### 1.2 浏览器支持

- Chrome 90+（推荐）
- Firefox 88+
- Safari 14+
- Edge 90+

## 2. 快速入门

### 2.1 登录系统

1. 打开浏览器，访问系统地址
2. 输入用户名和密码

## 3. 工作台

### 3.1 数据卡片

工作台顶部展示 4 项关键数据：

| 卡片 | 说明 |
|------|------|
| 今日签约 | 当日新增签约数量 |
| 本月业绩 | 当月累计签约金额 |
| 待跟进客户 | 需要跟进的客户数量 |
| 待审批 | 待处理的审批事项数 |

每个卡片带有环比趋势箭头（↑ 上升 / ↓ 下降）。

### 3.2 AI 洞察

系统会自动推送 AI 分析的业务洞察，分为三个等级：

- **蓝色（提示）**：一般性信息提醒
- **橙色（警告）**：需要关注的风险
- **红色（紧急）**：需要立即处理的问题

### 3.3 AI 巡检告警

系统每天 9:00 自动执行 AI 巡检，覆盖：

- **客户跟进超期**：超过 7 天未跟进的客户
- **佣金结算超期**：超过 30 天待结算的佣金
- **合同到期提醒**：30 天内签约满一年的客户
- **回款逾期**：已过应付日期的回款

进入 **监控 → AI 巡检** 页面查看告警列表，支持筛选、批量处理。

### 3.4 角色化门户

根据用户角色，工作台动态显示不同的功能入口：

- **销售顾问**：客户跟进、带看登记、佣金查询
- **管理者**：团队概览、审批中心、经营分析
- **HR**：人事管理、请假审批
- **管理员**：系统配置、用户管理

## 4. 客户管理

### 4.1 客户列表

- 支持按状态筛选（新客户/跟进中/已预约/已到访/已认购/已签约/流失）
- 支持关键词搜索客户姓名、电话
- 列表显示客户基本信息、意向度、最后跟进时间

### 4.2 新增客户

1. 点击「新增客户」按钮
2. 填写客户基本信息（姓名、电话、来源等）
3. 系统自动进行判客检查（避免重复录入）
4. 填写意向信息（意向项目、预算范围、面积需求）
5. 保存后自动分配给当前登录的销售顾问

### 4.3 客户跟进

1. 进入客户详情页
2. 在「跟进记录」区域点击「新增跟进」
3. 选择跟进方式（电话/微信/面谈/带看）
4. 填写跟进内容
5. 设置下次跟进时间
6. 系统自动更新客户的「最后跟进时间」

### 4.4 客户状态转换

客户状态流转：新客户 → 跟进中 → 已预约 → 已到访 → 已认购 → 已签约

每次状态变更都会记录在跟进历史中。

## 5. 项目管理

### 5.1 项目信息

- 维护楼盘基本信息（名称、地址、开发商、总户数等）
- 上传项目图片（效果图、户型图、区位图）
- 设置项目状态（在售/待售/售罄）

### 5.2 房源管理

- 按栋/单元/楼层管理房源
- 维护房源面积、单价、总价
- 标记房源状态（可售/已认购/已签约/已锁定）

### 5.3 通用项目管理（任务与协作）

通用项目管理用于**任务协作**（与 5.1 楼盘项目区分）：创建项目空间、拆解任务、看板/甘特/日历多视图、里程碑与依赖。

- **项目列表**：入口「协作办公 → 项目管理」。支持「我创建的」「我参与的」「全部」筛选，按状态（未开始/进行中/已暂停/已完成/已归档）筛选。可新建项目（名称、描述、负责人、计划起止日期、可见范围、主题色）。
- **项目详情**：点击项目进入，Tab 包括：
  - **任务列表**：表格展示任务（标题、执行人、状态、优先级、进度、截止日），支持新建任务、子任务、编辑/删除。任务可设置指派人、状态（待办/进行中/已完成）、进度百分比、计划开始/截止日期、预估/实际工时。
  - **看板**：按状态分列展示任务卡片，拖拽卡片可变更状态（与任务列表联动）。
  - **甘特图**：按时间轴展示任务条与依赖关系，支持查看进度与起止日期。
  - **日历**：月历视图展示任务截止日与里程碑，支持上/下月切换。
  - **里程碑**：项目级关键节点，可新建/编辑/标记完成，目标日期会在日历与甘特图中展示。
  - **设置**：编辑项目基本信息；管理成员（添加/移除、角色：负责人/管理员/成员）；可见范围（仅成员/本部门/全公司）。
- **任务依赖**：在任务详情中可添加「前置任务」，系统支持完成-开始（fs）等类型，并防止循环依赖。
- **工作台**：「我的任务」卡片展示当前用户被指派的任务（逾期/今日截止/进行中等），点击可跳转项目管理；应用入口提供「项目管理」快捷进入。
- **逾期与提醒**：任务逾期或里程碑临近到期时，系统会通过 Heartbeat 巡检生成告警并通知相关人员；全局搜索支持按「项目」或「任务」范围检索。

## 6. 佣金管理

### 6.1 佣金录入

1. 进入佣金管理页面
2. 选择关联客户和项目
3. 填写佣金信息（佣金比例、预计金额）
4. 关联联盟商（如有）
5. 提交审批

### 6.2 佣金结算

- 查看待结算佣金列表
- 确认结算金额
- 提交结算审批
- 审批通过后自动更新状态

### 6.3 佣金统计

经营分析大屏中提供佣金统计图表：
- 应结 vs 已结对比柱状图
- 按月/季度的佣金趋势

### 7.1 回款计划

## 8. 即时通讯

### 8.1 单聊

- 从联系人列表发起私聊
- 支持发送文本、图片、文件
- 图片可直接粘贴发送
- 文件可拖拽发送

### 8.2 群聊

- 创建群组，添加成员
- @提及特定成员（输入 @ 符号弹出选择列表）
- 群公告、群文件管理

### 8.3 消息功能

| 功能 | 操作方式 |
|------|----------|
| 消息撤回 | 右键点击消息 → 撤回（2 分钟内） |
| 消息引用 | 右键点击消息 → 引用回复 |
| 消息搜索 | 顶部搜索框输入关键词 |
| 桌面通知 | 自动弹出浏览器通知（需授权） |

### 8.4 消息优先级

系统自动根据发送人角色和关键词计算消息优先级：

- 🔥 **紧急消息**：红色高亮显示
- ⭐ **特别关注**：置顶区域展示
- 支持手动标记会话分类（关注/加星/静音）

### 8.5 待办提取

在聊天中发现任务时，可一键提取为待办事项：

1. 复制聊天消息内容
2. 进入待办管理页面
3. 在「消息待办提取」区域粘贴消息
4. 系统自动识别任务内容和优先级
5. 确认后添加到待办列表

## 9. 智能表格

入口：**协作办公 → 智能表格**。

### 9.1 创建表格

两种方式创建：

**方式一：空白创建**
1. 点击「创建表格」
2. 输入名称和描述
3. 逐个添加字段

**方式二：从模板创建**
1. 点击「从模板创建」
2. 选择适合的行业模板（客户跟进、楼盘对比、佣金结算等）
3. 输入表格名称
4. 系统自动创建包含预设字段和示例数据的表格

### 9.2 字段类型

支持 14 种字段类型：

| 类型 | 说明 | 示例 |
|------|------|------|
| 文本 | 单行/多行文字 | 客户姓名 |
| 数字 | 数值 | 面积、金额 |
| 日期 | 日期选择 | 签约日期 |
| 日期时间 | 精确到时分 | 带看时间 |
| 选择 | 下拉选择 | 跟进状态 |
| 多选 | 多项选择 | 客户标签 |
| 复选框 | 是/否 | 是否签约 |
| 评分 | 1-5 星评分 | 意向度 |
| 进度 | 百分比进度条 | 回款进度 |
| 成员 | 团队成员 | 置业顾问 |
| 电话 | 电话号码 | 联系电话 |
| 邮箱 | 邮箱地址 | 邮箱 |
| 链接 | URL 链接 | 相关链接 |
| 图片 | 图片上传 | 房源照片 |

### 9.3 多种视图

**表格视图**（默认）
- 类 Excel 的网格视图
- 双击单元格直接编辑
- 支持排序、筛选、分组

**看板视图**
- 按选择字段分组为列
- 拖拽卡片在列间移动
- 适合管理流程状态

**甘特图视图**
- 选择开始/结束日期字段
- 时间轴上展示任务条
- 适合项目进度跟踪

**日历视图**
- 月历网格展示
- 按日期字段映射记录
- 直观查看日程安排

### 9.4 模板管理

进入 **智能表格 → 表格模板管理**：

- 浏览系统预置的 8 个行业模板
- 创建自定义模板
- 导入/导出模板（JSON 格式）
- 从模板快速创建表格

预置模板清单：
1. 客户跟进记录表
2. 楼盘对比分析表
3. 佣金结算跟踪表
4. 带看登记表
5. 回款跟踪表
6. 签约台账表
7. 渠道拓客登记表
8. 日常巡盘记录表

## 11. 销售工具箱

### 11.1 经营分析大屏

进入 **经营分析** 查看：

- 4 张数据概览卡片（签约额、回款、佣金、去化率）
- 销售额趋势图（日/周/月维度）
- 项目去化率柱状图
- 回款进度饼图
- 佣金统计对比图
- AI 经营洞察建议

### 11.2 销售质检

系统每月自动对全员进行 6 维度销售能力评估：

| 维度 | 说明 |
|------|------|
| 沟通能力 | 与客户的沟通质量 |
| 需求挖掘 | 识别客户真实需求 |
| 异议处理 | 应对客户疑虑的能力 |
| 成交能力 | 推动签约转化 |
| 跟进频率 | 客户跟进的及时性 |
| 响应速度 | 回复客户的时效性 |

查看方式：
- ECharts 雷达图展示能力画像
- 综合评分 + AI 评语和改进建议
- 销售排行榜
- 评分趋势折线图

### 11.3 AI 日报

系统每晚 22:00 自动生成工作日报，包含：

- 当日客户跟进数
- 新增客户数
- 成交情况
- 待办完成率
- AI 分析建议

也可手动选择日期生成日报。

### 11.4 场景模板库

提供 4 类预置话术模板：

1. **楼盘推介纪要**：带看后的推介报告
2. **客户需求分析**：深度需求分析模板
3. **竞品对比报告**：多维度竞品分析
4. **项目踩盘报告**：踩盘调研模板

使用方式：选择模板 → 填写字段 → 生成文档 → 一键复制

### 11.5 销售日历

- 月/周/日三种视图
- 快捷创建日程事件
- 事件类型颜色标识
- 提醒配置
- 团队日程共享

### 11.6 自动化规则

预置 4 条业务规则：

1. 客户超 7 天未跟进自动提醒
2. 合同到期前 30 天预警
3. 新客户按区域自动分配
4. 成交自动通知管理者

支持自定义规则：设置触发条件和执行动作。

## 12. AI 智能助手

### 12.1 智能对话

在 AI 助手页面与系统对话：

- 询问业务数据（"这个月签了几单？"）
- 生成分析报告（"帮我分析一下客户转化率"）
- 获取工作建议（"这个客户应该怎么跟进？"）

支持多轮对话，系统会记住上下文。

### 12.2 AI 记忆

系统自动积累关于客户、项目、市场的知识记忆：

- **L1 短期记忆**：近期对话信息
- **L2 长期记忆**：重要经验和客户画像

记忆会自动精炼和清理，确保 AI 回答始终准确。

### 12.3 知识库

上传项目资料文档（PDF、Word、Excel），AI 可基于知识库回答问题：

1. 进入知识库管理页面
2. 上传文档
3. 系统自动解析和向量化
4. 在 AI 对话中引用知识库内容

## 15. 常见问题

### Q: 忘记密码怎么办？

联系系统管理员重置密码。

### Q: 搜索功能没有结果？

- 检查搜索关键词是否正确
- 全局搜索支持客户名、项目名等，不支持纯数字搜索
- 中文搜索至少输入 2 个字符

### Q: 如何导出数据？

在对应的列表页面，点击「导出」按钮即可导出 Excel。

### Q: AI 助手回答不准确？

- 尝试更具体地描述问题
- 上传相关文档到知识库
- 反馈给系统管理员调整 AI 配置

### Q: 消息通知没有声音？

- 检查浏览器是否允许通知权限
- 检查系统声音设置
- 确保浏览器标签页未静音

### Q: 如何查看操作日志？

进入 **系统管理 → 操作日志** 页面，可按操作类型、时间范围筛选。
