## 四、安装包生成测试

### 4.1 电脑端安装包

#### 方案一：Electron 打包（推荐）

**步骤**:
```bash
# 1. 安装 Electron
cd pengcheng-ui
npm install --save-dev electron electron-builder

# 2. 创建 electron-builder.config.js
# 3. 添加打包脚本
"scripts": {
  "electron:dev": "electron .",
  "electron:build": "electron-builder"
}

# 4. 打包
npm run electron:build
```

**输出**:
- Windows: `MasterLife-Setup.exe`
- macOS: `MasterLife.dmg`
- Linux: `MasterLife.AppImage`

**状态**: ⚠️ **需要额外配置**

#### 方案二：PWA 离线应用（轻量）

**步骤**:
```bash
# 1. 添加 PWA 支持
npm install vite-plugin-pwa

# 2. 配置 vite.config.ts
# 3. 构建
npm run build
```

**输出**:
- 可安装到桌面
- 支持离线使用
- 自动更新

**状态**: ✅ **推荐方案**

#### 方案三：Web 应用（最简单）

**步骤**:
```bash
# 1. 构建
npm run build

# 2. 部署到 Nginx
# 3. 用户通过浏览器访问
```

**状态**: ✅ **立即可用**

## 五、独立部署测试

### 5.1 Docker 一键部署 ✅

**部署命令**:
```bash
# 1. 克隆项目
git clone <repo-url>
cd pengcheng-admin

# 2. 配置环境变量
cp .env.example .env
# 编辑 .env 配置密码和密钥

# 3. 启动服务
docker-compose up -d

# 4. 查看状态
docker-compose ps

# 5. 访问服务
# 前端：http://localhost
# API 文档：http://localhost:8080/api/doc.html
```

**启动的服务**:
| 服务 | 端口 | 说明 |
|------|------|------|
| Nginx | 80/443 | 前端 + 反向代理 |
| App | 8080 | Spring Boot 后端 |
| MySQL | 3306 | 数据库 |
| Redis | 6379 | 缓存 |
| PostgreSQL | 5432 | 向量库 |
| MinIO | 9000/9001 | 文件存储 |
| kkFileView | 8012 | 文件预览 |
| OnlyOffice | 8443 | 在线编辑 |

**状态**: ✅ **立即可用**

### 5.2 手动部署

**后端部署**:
```bash
# 1. 编译
mvn clean package -DskipTests

# 2. 运行
java -jar pengcheng-starter/target/*.jar \
  --spring.profiles.active=prod
```

**前端部署**:
```bash
# 1. 构建
npm run build

# 2. 部署到 Nginx
cp -r dist/* /usr/share/nginx/html/
```

**状态**: ✅ **立即可用**
