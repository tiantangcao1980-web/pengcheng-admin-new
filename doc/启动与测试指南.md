## 八、常见问题

### Q1: 服务启动失败？

**答**: 
1. 检查端口是否被占用：`lsof -i :8080`
2. 检查 Docker 是否运行：`docker ps`
3. 查看错误日志：`docker-compose logs`

### Q2: 前端无法访问？

**答**:
1. 检查 Nginx 容器：`docker-compose ps nginx`
2. 查看 Nginx 日志：`docker-compose logs nginx`
3. 清除浏览器缓存

### Q3: AI 功能不可用？

**答**:
1. 检查 API 密钥是否配置
2. 查看后端日志：`docker-compose logs -f app`
3. 确认网络连接正常

### Q4: 数据库连接失败？

**答**:
1. 检查 MySQL 容器状态
2. 确认 .env 中密码配置正确
3. 等待 MySQL 完全启动（约 30 秒）
