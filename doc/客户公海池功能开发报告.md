## 一、功能概述

客户公海池是房地产 CRM 系统中的核心功能，用于管理客户资源的公平分配和高效利用。

### 1.1 核心概念

| 概念 | 说明 |
|------|------|
| **公海池** | 公共客户资源池，所有销售人员可查看和领取 |
| **私海池** | 销售人员的私有客户池，拥有保护期 |
| **保护期** | 领取客户后的一定期限（默认 3 天），期间不会被回收 |
| **自动回收** | 满足条件的私海客户自动回收到公海池 |

### 1.2 回收规则

| 规则类型 | 默认阈值 | 说明 |
|---------|---------|------|
| 无跟进回收 | 7 天 | 超过 7 天无跟进记录的客户 |
| 未到访回收 | 30 天 | 已报备状态超过 30 天未到访的客户 |
| 已成交保护 | 永久 | 已成交客户不参与回收 |

### 4.1 自动回收逻辑

```java
// CustomerPoolService.java
public int recycleToPublicPool() {
    LocalDateTime now = LocalDateTime.now();
    LocalDateTime noFollowThreshold = now.minusDays(noFollowDays);
    LocalDateTime noVisitThreshold = now.minusDays(noVisitDays);

    // 查询私海池中满足回收条件的客户
    LambdaQueryWrapper<Customer> wrapper = new LambdaQueryWrapper<>();
    wrapper.eq(Customer::getPoolType, POOL_PRIVATE)
            .ne(Customer::getStatus, STATUS_DEAL) // 已成交不回收
            .and(w -> w
                // 条件 1：超过无跟进天数
                .lt(Customer::getLastFollowTime, noFollowThreshold)
                .or()
                // 条件 2：状态为已报备且超过未到访天数
                .nested(inner -> inner
                    .eq(Customer::getStatus, STATUS_REPORTED)
                    .lt(Customer::getCreateTime, noVisitThreshold)
                )
            );

    // 分批处理，每批 100 条
    // 使用乐观锁（基于 updateTime）避免并发冲突

}
```

### 4.2 定时任务配置

```java
// CustomerPoolRecycleTask.java
@Component("customerPoolRecycleTask")
public class CustomerPoolRecycleTask {
    private final CustomerPoolService customerPoolService;

    public void execute() {
        log.info("开始执行公海池自动回收任务...");
        int recycled = customerPoolService.recycleToPublicPool();
        log.info("公海池自动回收任务完成，共回收 {} 个客户", recycled);
    }
}
```

### 4.3 批量领取逻辑

```java
// CustomerController.java
@PostMapping("/pool/claim")
public Result<Void> claimFromPublicPool(@RequestBody PoolClaimDTO dto) {
    for (Long customerId : dto.getCustomerIds()) {
        customerPoolService.claimFromPublicPool(customerId, dto.getUserId());
    }
    return Result.ok();
}

// CustomerPoolService.java
@Transactional
public void claimFromPublicPool(Long customerId, Long userId) {
    Customer customer = customerMapper.selectById(customerId);
    
    customer.setPoolType(POOL_PRIVATE);
    customer.setProtectionExpireTime(now.plusDays(DEFAULT_PROTECTION_DAYS));
    customer.setLastFollowTime(now);
    customer.setCreatorId(userId);
    
    customerMapper.updateById(customer);
}
```
