## 五、部署与使用

### 5.1 一键部署

```bash
# 1. 克隆项目
git clone <repo-url>
cd pengcheng-admin

# 2. 配置环境变量
cp .env.example .env

# 3. 启动服务
docker-compose up -d

# 4. 访问
# 前端：http://localhost
# API 文档：http://localhost:8080/api/doc.html
```

### 5.2 安装包生成

**电脑端**:
- Web 版：直接访问 URL
- PWA 版：浏览器安装
- Electron：可选打包

**移动端**:
- Android APK：HBuilderX 云打包
- iOS IPA：App Store 发布
- 微信小程序：微信审核发布

### 5.3 员工使用

**难度**: ⭐⭐ 简单
